Over TaalMap

TaalMap is een web applicatie bedoeld als hulpmiddel bij het volgen van een vreemdetaalstudie, bijv. bij een een Volks Universiteit.

TaalMap is geschreven in JavaScript/TypeScript en maakt gebruik van de volgende Open Source bouwstenen:

  • Angular 2.0
  • Bootstrap v4.0
  • ng2-bootstrap
  • Font Awesome
  • marked

Voor de tekstbestanden en de woordenboekfunctie maakt TaalMap gebruik van een back-end server die is opgebouwd met de volgende Open Source software:

  • Node.js
  • express.js
  • MongoDB
  • Mongoose
